# vita-libvita2d-game
a small, nameless game implementation for the ps vita wip

(demo on youtube)[https://www.youtube.com/watch?v=3YxhuEGMrEI]

## build

dependencies: 
- vitasdk
- unix environment
- psvita with 3.60 henkaku

## bash:

create a vpk, ready to install on a henkaku 3.60 psvita:

``make clean && make``
